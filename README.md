# Koifit - Workout Tracker MVP

## Quick Start

1. **Install dependencies**:
   ```bash
   uv sync
   ```

2. **Initialize the database**:
   ```bash
   uv run python init_db.py
   ```

3. **Run the server**:
   ```bash
   uv run uvicorn main:app --reload
   ```

4. **Access the app**:
   Open `http://localhost:8000` in your browser
